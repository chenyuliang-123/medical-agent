## 技术细节

### 时间处理逻辑

```python
# 1. 只有时间（HH:MM）
"09:00" → 今天 09:00:00（如果已过，则明天 09:00:00）

# 2. 日期+时间（YYYY-MM-DD HH:MM）
"2025-11-05 09:00" → 2025-11-05 09:00:00

# 3. 完整时间（YYYY-MM-DD HH:MM:SS）
"2025-11-05 09:00:00" → 2025-11-05 09:00:00
```
